## 개요

스마일게이트 언리얼 개발자 양성 프로그램 'UNSEEN'에서 진행한 1인 프로젝트 'Collapsing'입니다.  
언리얼 엔진 5를 활용해 실감나는 3D Running Game을 제작했습니다.  
간단한 조작으로 누구나 쉽게 할 수 있지만, 다양한 요소로 긴박함을 불어넣는 것을 목표로 했습니다.
